# Telegram Notify

Allows the assistant to send messages to a specific Telegram chat ID.

## Send Message

```bash
python3 nanobot/skills/telegram_notify/notify.py "<chat_id>" "<message_content>"
```

*Replace `<chat_id>` and `<message_content>` with the target chat ID and the message text.*
